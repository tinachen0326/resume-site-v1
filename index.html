轉職動機
在過去的工作中，我擅長行銷素材設計與平面排版。但隨著對使用者行為與網頁互動的興趣日益濃厚，我發現自己更嚮往將設計概念實際轉化為可操作介面的過程。這促使我重新思考職涯方向，決心朝向前端工程領域發展，期望能透過程式技術實現更好的使用者體驗，並在產品開發中發揮更大的影響力。
進修過程
為實現轉職目標，我參與文化大學 X 程式驅動全端工程師養成班，系統性學習全端技術，包含 HTML/CSS、JavaScript、Vue.js 等現代前端框架，以及後端 API 整合。課程中實作多項整合 UI 設計與工程開發的專案，讓我更明確發現自己在前端開發、元件設計、使用者介面邏輯實作等方面具有高度執行能力與學習熱忱。
